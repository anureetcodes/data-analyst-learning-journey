🧠 **Key Concepts Learned**

**What is Data Analysis?**

-process of identifying , cleaning , tranforming ,modeling data to discover meaningful information.
- where analytics acts as a bridge between raw data and a fact based data .
- Supporting business decision-making using evidence.

**Components of analytics**
  
  - descriptive analytics: helps answer question, what has happened?
  - diagnostic  analytics :  answer the question, why events happened?
  - priscriptive analytics:  helps answer which action sshould be taken to achieve a target
  - predictive analytics: helps answer what will happen?
    
 **Role of a Data Analyst**
 
- collect data 
- data prepartion: profiling, cleaning , transforming 
- Model: connect separate infromation and find relationship between them 
- Visualize : create  well deisgn report that tells compelling story about data 
- Analyze : find actionable insights and patterns
- Manage: managing power BI assets
- 
 **Data Analysis Process**
  
- Ask the business question
- collect data
- clean and prepare data
- visualize results
- analyze the data
- share insights

**Importance of Data Storytelling**

Numbers alone don't create impact. A data analyst communicates findings through clear visuals and meaningful business stories.
he should be able to look at the  data and tell what data is trying to tell
